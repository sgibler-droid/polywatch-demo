# PolyWatch Demo

Public static demo for PolyWatch, a Polymarket wallet intelligence dashboard.

Live GitHub Pages version:
https://sgibler-droid.github.io/polywatch-demo/

Netlify setup:
- Import this repo in Netlify: https://github.com/sgibler-droid/polywatch-demo
- Build command: empty
- Publish directory: `.`

This public version uses the bundled `snapshot.json` rather than the live Mac Mini API.
